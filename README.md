# Smart Car Parking System using IoT & Machine Learning

## 🔹 Overview
This project is an IoT-based smart parking system that detects vehicle presence and displays real-time parking availability.

## 🔹 Features
- Real-time parking slot monitoring
- Dashboard UI for slot visualization
- Parking history tracking
- Machine Learning model for occupancy detection

## 🔹 Technologies Used
- ESP32 / Arduino
- Python (Machine Learning)
- MySQL (Database)
- HTML/CSS (Dashboard UI)

## 🔹 Results
- Confusion Matrix for model evaluation
- Training & Validation Accuracy Graphs
- Real-time parking dashboard
